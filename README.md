# Word Keeper
## Used by vue-cli with vue3 package

## Project setup
```
yarn
```
## Fill in the .env file
```
Add API key of wordsapiv1.p.rapidapi.com to .env (or create .env.local for add to gitignore)
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
